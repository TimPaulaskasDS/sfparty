# Agent Notes

## Package Manager

**This project uses `bun`, not `npm`.**

When running commands, use:
- `bun run build` instead of `npm run build`
- `bun test` instead of `npm test`
- `bun install` instead of `npm install`
- `bun add <package>` instead of `npm install <package>`
- `bunx` instead of `npx` when needed

## Quick Reference

- **Build**: `bun run build`
- **Test**: `bun test`
- **Install dependencies**: `bun install`
- **Run CLI**: `bun run dist/index.js <command>`
